## About
wp-log is a tool for finding debug-log.

## Features

- Easy to Install
- Easy to Use
- Designed In Bash
- Easy To Execute 

## How to Install

### Prerequisites

- Git

### Installation

- Clone the Repo
  ```bash
  git clone https://github.com/BiswajeetRay7/wp-log.git
  ```
- Open cloned directory
  ```bash
  cd wp-log && chmod +x wp-log.sh 
  ```
- Now run tool
  ```bash
  ./wp-log.sh target.txt
  ```
> Now it will run with all sites.
